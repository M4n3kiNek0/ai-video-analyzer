# Screenshots

This folder contains screenshots for the README documentation.

## Required Screenshots

Please capture and add the following screenshots:

### 1. `dashboard.png`
- **Page:** Main dashboard (http://localhost:3000)
- **Content:** Show the video library with several video cards
- **Recommended size:** 1200x800px

### 2. `upload.png`
- **Page:** Upload page (http://localhost:3000/upload)
- **Content:** Show the drag & drop interface with a file selected
- **Recommended size:** 800x600px

### 3. `analysis.png`
- **Page:** Video detail page (http://localhost:3000/videos/[id])
- **Content:** Show video player with transcript sidebar visible
- **Recommended size:** 1200x800px

### 4. `processing.png`
- **Page:** Video detail page → Logs tab
- **Content:** Show the real-time processing terminal
- **Recommended size:** 1200x600px

## Tips for Great Screenshots

1. **Use a clean browser** – Hide bookmarks bar and extensions
2. **Consistent sizing** – Use DevTools to set viewport dimensions
3. **Real data** – Use actual analyzed videos, not empty states
4. **Dark mode** – The app uses a dark theme, capture it that way
5. **High DPI** – If possible, capture at 2x resolution for crisp images
